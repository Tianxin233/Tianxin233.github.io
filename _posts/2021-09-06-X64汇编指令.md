# X64汇编指令

# mov

```汇编
`mov  dest ，src`
```

- 将src的值赋值给dest，类似于dest=src

```
[地址值] 
```

- 中括号[]里面存放的都是内存地址

```
int a = 10; 
mov dword ptr [ebp-8],0Ah   //将0AH就是10赋值给 ebp-8也就是a所在的内存空间
int b = a + 1; 			
mov eax,dword ptr [ebp-8]  //dword 双字4个字节 ptr是指针 dword ptr [ebp-8] 将内存地址ebp-8中的双字节赋值给eax
add eax,1  
mov dword ptr [ebp-14h],eax  
```
